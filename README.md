Registro de Usuario con Validación en JavaScript
Descripción

Este proyecto consiste en un formulario de registro de usuario desarrollado con HTML, CSS y JavaScript que realiza validaciones del lado del cliente.
El sistema verifica los datos ingresados y muestra mensajes de error cuando los campos no cumplen los requisitos.

Funcionalidades

Validación de nombre (mínimo 3 caracteres).

Validación de correo electrónico con formato correcto.

Validación de contraseña (mínimo 8 caracteres, una mayúscula y un número).

Confirmación de contraseña.

Validación de teléfono opcional (solo números).

Mensajes de error en cada campo.

Indicador de fortaleza de contraseña (débil, regular, buena, fuerte).

Tecnologías utilizadas

HTML5

CSS3

JavaScript
